# 基本面数据 > 股票增发

## 接口: stock_add_stock

### 描述
新浪财经-发行与分配-增发

### 输入参数
输入参数
| 名称     | 类型  | 描述                    |
|--------|-----|-----------------------|
| symbol | str | symbol="600004"; 股票代码 |

### 输出参数
输出参数
| 名称         | 类型     | 描述 |
|------------|--------|----|
| 公告日期       | object | -  |
| 发行方式       | object | -  |
| 发行价格       | object | -  |
| 实际公司募集资金总额 | object | -  |
| 发行费用总额     | object | -  |
| 实际发行数量     | object | -  |
