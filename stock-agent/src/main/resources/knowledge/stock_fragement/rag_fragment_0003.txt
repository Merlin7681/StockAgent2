# A股 > 股票市场总貌 > 深圳证券交易所 > 地区交易排序

## 接口: stock_szse_area_summary

### 描述
深圳证券交易所-市场总貌-地区交易排序

### 输入参数
输入参数
| 名称   | 类型  | 描述                |
|------|-----|-------------------|
| date | str | date="202203"; 年月 |

### 输出参数
输出参数
| 名称    | 类型      | 描述      |
|-------|---------|---------|
| 序号    | int64   | -       |
| 地区    | object  | -       |
| 总交易额  | float64 | 注意单位: 元 |
| 占市场   | float64 | 注意单位: % |
| 股票交易额 | float64 | 注意单位: 元 |
| 基金交易额 | float64 | 注意单位: 元 |
| 债券交易额 | float64 | 注意单位: 元 |
