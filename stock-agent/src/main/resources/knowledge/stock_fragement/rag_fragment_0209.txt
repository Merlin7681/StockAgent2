# 基本面数据 > 终止/暂停上市-深证

## 接口: stock_info_sz_delist

### 描述
深证证券交易所终止/暂停上市股票

### 输入参数
输入参数
| 名称     | 类型  | 描述                                              |
|--------|-----|-------------------------------------------------|
| symbol | str | symbol="终止上市公司"; choice of {"暂停上市公司", "终止上市公司"} |

### 输出参数
输出参数
| 名称     | 类型     | 描述  |
|--------|--------|-----|
| 证券代码   | object | -   |
| 证券简称   | object | -   |
| 上市日期   | object | -   |
| 终止上市日期 | object | -   |
