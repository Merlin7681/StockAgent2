# 基本面数据 > 名称变更-深证

## 接口: stock_info_sz_change_name

### 描述
深证证券交易所-市场数据-股票数据-名称变更

### 输入参数
输入参数
| 名称     | 类型  | 描述                                        |
|--------|-----|-------------------------------------------|
| symbol | str | symbol="全称变更"; choice of {"全称变更", "简称变更"} |

### 输出参数
输出参数
| 名称    | 类型     | 描述  |
|-------|--------|-----|
| 变更日期  | object | -   |
| 证券代码  | object | -   |
| 证券简称  | object | -   |
| 变更前全称 | object | -   |
| 变更后全称 | object | -   |
