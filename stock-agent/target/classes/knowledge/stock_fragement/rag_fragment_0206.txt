# 基本面数据 > 股票列表-上证

## 接口: stock_info_sh_name_code

### 描述
上海证券交易所股票代码和简称数据

### 输入参数
输入参数
| 名称     | 类型  | 描述                                               |
|--------|-----|--------------------------------------------------|
| symbol | str | symbol="主板A股"; choice of {"主板A股", "主板B股", "科创板"} |

### 输出参数
输出参数
| 名称   | 类型     | 描述  |
|------|--------|-----|
| 证券代码 | object | -   |
| 证券简称 | object | -   |
| 公司全称 | object | -   |
| 上市日期 | object | -   |
