# A股 > 股票市场总貌 > 深圳证券交易所 > 证券类别统计

## 接口: stock_szse_summary

### 描述
深圳证券交易所-市场总貌-证券类别统计

### 输入参数
输入参数
| 名称   | 类型  | 描述                                  |
|------|-----|-------------------------------------|
| date | str | date="20200619"; 当前交易日的数据需要交易所收盘后统计 |

### 输出参数
输出参数
| 名称   | 类型      | 描述      |
|------|---------|---------|
| 证券类别 | object  | -       |
| 数量   | int64   | 注意单位: 只 |
| 成交金额 | float64 | 注意单位: 元 |
| 总市值  | float64 | -       |
| 流通市值 | float64 | -       |
