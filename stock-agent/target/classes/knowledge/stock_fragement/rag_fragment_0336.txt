# 股票热度 > 热搜股票

## 接口: stock_hot_search_baidu

### 描述
百度股市通-热搜股票

### 输入参数
输入参数
| 名称     | 类型  | 描述                                              |
|--------|-----|-------------------------------------------------|
| symbol | str | symbol="A股"; choice of {"全部", "A股", "港股", "美股"} |
| date   | str | date="20250616"                                 |
| time   | str | time="今日"; choice of {"今日", "1小时"}              |

### 输出参数
输出参数
| 名称    | 类型     | 描述 |
|-------|--------|----|
| 名称/代码 | object | -  |
| 涨跌幅   | object | -  |
| 综合热度  | int64  | -  |
