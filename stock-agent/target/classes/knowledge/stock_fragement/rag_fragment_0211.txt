# 基本面数据 > 暂停/终止上市-上证

## 接口: stock_info_sh_delist

### 描述
上海证券交易所暂停/终止上市股票

### 输入参数
输入参数
| 名称     | 类型  | 描述                                         |
|--------|-----|--------------------------------------------|
| symbol | str | symbol="全部"; choice of {"全部", "沪市", "科创板"} |

### 输出参数
输出参数
| 名称     | 类型     | 描述  |
|--------|--------|-----|
| 公司代码   | object | -   |
| 公司简称   | object | -   |
| 上市日期   | object | -   |
| 暂停上市日期 | object | -   |
